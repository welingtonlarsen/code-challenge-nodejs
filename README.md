## Local prerequisites
- Node.js (16.14.0 is recommended)
- Docker

## Execution
  ### Local infra
    docker-compose up -d

  ### Database migration (necessary only at the first time)
      yarn migration
  
  ### Development environment app start
    yarn dev
  
  ### Run unit tests
    yarn test
